# NeMo Training Service

On-demand NVIDIA NeMo container for training and fine-tuning audio models on the DGX Spark.

## Quick Start

```bash
# Pull image (first time only — large download)
docker compose --profile training pull nemo

# Interactive shell
docker compose --profile training run --rm nemo bash

# Check NeMo version
docker compose --profile training run --rm nemo python -c "import nemo; print(nemo.__version__)"
```

## Data Layout

Place training data here on the host before starting the container:

```
/home/jdn/Data/NeMo/
├── audio/
│   ├── <class_label_1>/
│   │   ├── sample_001.wav
│   │   └── ...
│   └── <class_label_2>/
│       └── ...
└── manifests/       # generated by prepare_data.py
    ├── train.json
    ├── val.json
    └── test.json

/home/jdn/Data/NeMoCheckpoints/   # model checkpoints saved here
```

Inside the container these map to `/workspace/data/` and `/workspace/checkpoints/`.

## Workflow

### Step 1: Prepare manifests

```bash
docker compose --profile training run --rm nemo \
    python /workspace/scripts/prepare_data.py \
    --audio-dir /workspace/data/audio \
    --manifest-dir /workspace/data/manifests
```

### Step 2: Fine-tune a pre-trained model

```bash
docker compose --profile training run --rm nemo \
    python /workspace/scripts/finetune_audio_classifier.py \
    --train-manifest /workspace/data/manifests/train.json \
    --val-manifest   /workspace/data/manifests/val.json \
    --num-classes    <N> \
    --epochs         20
```

Output: `/workspace/checkpoints/audio_classifier_finetuned.nemo`

### Step 3 (later): Train from scratch

See `scripts/train_audio_classifier.py` — implement once fine-tuning baseline is established.

## Notes

- `ipc: host` and `shm_size: 16g` are required for multi-GPU DataLoader workers.
- The container does not auto-restart; it is started on demand with `--profile training`.
- Install additional Python deps inside the container with `pip install <pkg>` (changes are
  lost on exit; add persistent deps to a custom Dockerfile layer if needed).
